About the Website

This website was developed as part of an educational project focused on data visualisation and web development. Its purpose is to demonstrate how appliance energy consumption data can be presented clearly and interactively using modern web technologies.

The content focuses on household electricity usage in Australia, using televisions as an example appliance category.

Project Objectives

The key objectives of this website are to:

  • Demonstrate the use of HTML, CSS, and JavaScript in a multi-page website
  • Practice version control using GitHub
  • Prepare a platform for future interactive data visualisations
  • Provide informative content related to appliance energy consumption

Educational Context

This project forms part of coursework for COS30045 - Data Visualisation and is intended for academic and demonstration purposes only. Energy values and statistics presented are indicative and may not reflect specific appliance models.
